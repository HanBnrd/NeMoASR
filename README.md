# NeMoASR

Automatic speech recognition with speaker diarisation on small GPUs

## Requirements
- Ubuntu
- Python 3.12+
- NVIDIA GPU 12GB+

## Setup
```bash
sudo apt update && sudo apt install ffmpeg
```
```bash
pip install git+https://github.com/HanBnrd/NeMoASR.git

```

## Usage
```bash
nemoasr myfile.mp3
```
